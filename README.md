# Produtos em C — Ponteiros e Memória

Programa em C que demonstra o uso de ponteiros, passagem por referência, structs, aritmética de ponteiros e alocação dinâmica de memória.

## Atividades

| # | Conceito |
|---|----------|
| 1 | Ponteiro para `float` — aumento de 10% via desreferência |
| 2 | Passagem por referência — função `aplicar_desconto` |
| 3 | Ponteiro para `struct` — operador `->` |
| 4 | Array estático — aritmética de ponteiros em `imprimir_produtos` |
| 5 | Alocação dinâmica — `malloc` e `free` |

## Como compilar e rodar

```bash
gcc -Wall -Wextra -o produtos produtos.c
./produtos
```

## Exemplo de entrada (Atividade 5)

```
Quantos produtos deseja cadastrar? 2
Preco do produto 201: 59.90
Preco do produto 202: 149.00
```
